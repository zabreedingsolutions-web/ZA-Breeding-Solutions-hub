# Za Breeding Solutions

## Project info

Local development for the Za Breeding Solutions site built with Vite + React.

## How to run locally

Ensure you have Node.js and npm installed.

```sh
git clone <YOUR_GIT_URL>
cd <YOUR_PROJECT_NAME>
npm install
npm run dev
```

## Stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Deploy

Build with `npm run build` and deploy the `dist` output to your static host.
